# Twój Organizer!-terminarz
## CodersCamp
Projekt 4. w ramach kursu CodersCamp (edycja letnia 2019).
## Backend
Kod API znajduje się na branchu *Backend*. 


API zostało zdeployowane przy pomocy Heroku pod następującym adresem: [https://api-terminarz.herokuapp.com](https://api-terminarz.herokuapp.com). 

## Frontend 
W folderze *frontend* znajdują się pliki webpacka, folder *docs* zawiera pliki na potrzeby demo. 
## Uruchamianie lokalnie
By uruchomić aplikację należy pobrać zawartość folderu *frontend* i uruchomić z użyciem komend:
```
npm install
npm start
```
Aplikację można uruchomić dla bazy lokalnej (z wykorzystaniem kodu z brancha *Backend*) lub API zewnętrznego.
